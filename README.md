Websocket-conn-go
===
An example application to understand Web socket protocol. A handy app to get started with.
## Use
- Clone 
  ```
  git clone https://github.com/anazibinurasheed/websocket-conn-go.git
  ```
- Download deps
  ```
  go mod tidy
  ```

- Run
  ```
  go run .
  ```
## Directory Hierarchy
```
|—— dockerfile
|—— go.mod
|—— go.sum
|—— index.html
|—— main.go
```
